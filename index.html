Chorus

Run a containerized AI engineering team that turns specs into tickets, worktrees, branches, and reviewable pull requests.

Run it yourself Use hosted infra

Open source, container first

Keep powerful coding agents away from your host machine.

Chorus runs Codex, Claude Code, and Gemini CLI with the permissions they need for autonomous work, but wraps that work in a container, isolated git worktrees, durable logs, quota awareness, and human PR review before anything reaches the base branch.

Launch locally
# macOS
./deploy/mac/run-container.sh start

# Linux
./deploy/linux/run-container.sh start

# Windows PowerShell
.\deploy\windows\run-container.ps1 start

BYOAI backends

Bring the AI subscriptions or API keys you already use. Chorus detects the installed CLIs and lets each agent use the right model.

Container isolation

Agent processes run with dangerous approval bypasses inside a Linux container instead of directly on your workstation.

PR-native workflow

Every ticket lands as a branch and pull request. Humans keep the merge decision, while Chorus tracks status back in the dashboard.

Managed option

Open source when you want control. Hosted when you want velocity.
